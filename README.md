# PixelPulse


Automatically changes your desktop wallpaper based on the time of day, weather, or battery level.

## Features
- Time-based wallpapers
- Weather integration
- Battery-aware themes
- Random mode
- Lightweight

## Installation
```bash
git clone https://github.com/yourusername/PixelPulse.git
cd PixelPulse
pip install -r requirements.txt
python main.py
```
